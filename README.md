# webpack-template webpack react项目模板
> 重要文件版本

> 最新模板 react-router4 webpack3 [看这里](https://github.com/mjzhang1993/react-router4-template)

## 1.编译并生成开发版本

> `npm run dev`

## 2.输出生产版本

> `npm run build`

## 3.本地运行开发版本

> `npm run run`

- 2.服务器地址在` webpack -- API.js` 中配置，另外生产版本需要添加 `output.publicPath`
